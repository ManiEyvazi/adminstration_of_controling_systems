# Выводы по проделанной работе

## Вывод

В ходе лабораторной работы были освоены:

- принципы работы SELinux и его режимов (enforcing, permissive, disabled);
- методы изменения состояния и восстановления контекстов безопасности;
- настройка SELinux для нестандартных каталогов веб-сервера;
- работа с переключателями (booleans) SELinux.

Результатом стало понимание механизмов принудительного контроля доступа и умение диагностировать проблемы, связанные с политиками безопасности Linux.
