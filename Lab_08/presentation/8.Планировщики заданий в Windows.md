## Планировщики Заданий

## Windows

Автоматизация рутинных операций


## Назначение Планировщиков Заданий

Автоматизация

Выполнение задач без ручного вмешательства.

```
Рутинные Операции
Ежедневное обслуживание системы, резервное копирование.
```
Безопасность

Обновления безопасности, сканирование на вирусы.

```
Оптимизация
Регулярная очистка дисков, дефрагментация.
```

#### Архитектура Службы Task Scheduler

```
Служба
Управляет запуском и мониторингом задач
```
```
Хранилище
Сохраняет определения и состояние задач
```
```
API / UI
Интерфейсы для создания и управления
задачами
```
Служба планировщика заданий является центральным компонентом, управляющим выполнением всех запланированных задач.


###### Графический Интерфейс: taskschd.msc

```
Оснастка MMC
Интуитивно понятный интерфейс для
создания и управления задачами.
```
- Создание базовых задач
- Настройка сложных сценариев
- Мониторинг состояния задач


## Триггеры Заданий

```
Расписание
Ежедневно, еженедельно, ежемесячно, однократно.
```
```
События
При наступлении определенного события в журнале.
```
```
Состояние Системы
При запуске, при простое, при подключении к сети.
```
```
Вход Пользователя
При входе любого или конкретного пользователя в
систему.
```

### Действия Заданий

Типы Действий

```
Определяют, что выполняет задача при
срабатывании триггера.
```
- Запуск программы: Выполнение
    исполняемых файлов или скриптов.
- Отправка Email: Уведомления по
    электронной почте (устарело).
- Вывод сообщения: Отображение
    текстовых сообщений пользователю.


## Командная Строка: Schtasks

```
schtasks /create /tn "MyTask" /tr "notepad.exe" /sc HOURLY
```
```
schtasks /query /tn "MyTask" /v
```
```
schtasks /change /tn "MyTask" /st 18:
```
```
schtasks /delete /tn "MyTask" /f
```
Гибкое Управление

```
Идеально для автоматизации и
скриптов.
```
- Создание, изменение, удаление
    задач.
- Запрос информации о задачах.
- Импорт/экспорт задач.
